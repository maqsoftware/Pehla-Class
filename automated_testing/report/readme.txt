Report section of Local Automated Testing System

- These scripts are used to generate the testing report.
- emailAutomation.py generates HTML based report on email.Add To and From email in this script and app password for the authentication.
- reportGeneration.py generate Excel report.
- extractVBA.py automatically extracts the implemented functionalities in functionsVBA.xlsm file (as binary file) which will be used in the generated excel report.
