onebillion Testing Scripts and resources

- waitExtraction.py covers all the test cases related to splash screen, wait for the extraction of OBB and the ready screen after extraction.
- gameSection covers all the test cases related to Game Section of the application.
- playGround.py covers all the test cases related to Playground Section of the application.
- images folder contains all the templates used in testing.
- expected_result folder contains all the expected result images of all the testcases.
